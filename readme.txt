This plugin adds Twitter/Weibo buttons to the bottom of every post.

== Description ==

This plugin does one thing, and that thing is to add Twitter/Weibo buttons to the end of each of your posts. This change happens on the fly, so if you deactivate or uninstall this plugin, the feature goes away. 

== Installation ==

1. Upload the `tweebo` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
